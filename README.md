# Tata Motors - Corporate Website Clone

✨ A high-fidelity and responsive enterprise landing page clone of Tata Motors. Showcasing an extensive automobile portfolio, interactive car showcases, and a premium automotive digital experience.

## 🔗 Live Demo
Check out the live project here: [108580-tatamotors.netlify.app](https://108580-tatamotors.netlify.app/)

---

## 🚀 Features

- **Automotive Portfolio**: Comprehensive sections for Tata's leading vehicle lineup including Nexon, Safari, Harrier, Curvv, Altroz, Tiago, Tigor, and Punch.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Interactive UI**: Custom hover effects, festive sales banners, and smooth transitions.
- **Multi-page Experience**: Includes functional layouts for:
  - **Landing Page**: Integrated showcase of all vehicle segments.
  - **Newsroom**: Corporate updates and press releases.
  - **Account**: User login and profile management interface.
  - **Car Details**: Deep-dive sections for specific vehicle features.
- **Asset Integration**: High-quality SVG icons and optimized automotive imagery.

---

## 🛠️ Tech Stack

- **HTML5**: Semantic markup for structural integrity.
- **CSS3**: Vanilla CSS for layouts (Flexbox/Grid), custom animations, and responsive styling.
- **Iconography**: Custom SVG icons and FontAwesome integration.
